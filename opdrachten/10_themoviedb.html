The Movie DB (JSONP)

  • Zoek een film op titel.
  • Van alle gevonden titels, verschijnt de titel en het id in keuzelijst.
    De code: <option value="id">title</option>
  • De keuzelijst haalt de detailgegevens op (poster_path, title, homepage en overview)
  • Toon deze gegevens in #detail.
  • LET OP: voor deze API heb je een key nodig (23afca60ebf72f8d88cdcae2c4f31866).
  • themoviedb API:
    http://docs.themoviedb.apiary.io/
  • Voorbeeld:
    Zoek titel "Harry Potter"
    Details van ID 12445
  • LET OP: gebruik in $.getJSON() niet callback=? maar wel callback=23afca60ebf72f8d88cdcae2c4f31866;
